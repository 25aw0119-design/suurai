作品概要

URL https://25aw0119-design.github.io/sinkyuuseisaku/index.html
目的

健康に関するコラムを設けることで、 ブランドの「健康的なイメージ」を効果的に伝え、 ユーザーに「安心感」や「信頼感」を与えられるように提案しました。

ターゲット

家族の健康と笑顔のために、工夫を絶やさない４０代。

担当

企画・デザイン・コーディング

制作期間

３ヶ月

使用ツール

Figma / Illustrator / Photoshop / HTML / CSS / JavaScript

デザイン

健康的で親しみやすい印象を与えるため、 白を基調とした配色を採用しました。 また、料理写真を大きく配置し、 健康コラムやレシピを見つけやすいレイアウトにすることで、 情報が自然と目に入るデザインを目指しました。

工夫した点

レシピだけではブランドの魅力が伝わりにくいという課題に対し、 健康に関するコラムを設けることで、 ブランドが大切にする「健康」という価値を自然に伝えられるよう工夫しました。 また、レシピ・季節の特集を回遊しやすい導線にすることで、 ユーザーが継続的に利用したくなるサイトを意識しました。

学んだこと

ユーザーに情報を届けるためには、 レシピを掲載するだけでなく、 ブランドの価値や世界観を体験として伝えることが重要だと学びました。 また、ユーザーが目的の情報へ迷わずたどり着けるよう、 情報設計や導線設計を考えることの大切さを実感しました。

完成デザイン

健康コラムや季節の特集を通して、 ブランドの価値や食の楽しさを体験できるサイトを制作しました。
